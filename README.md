# Nderfaqe web e nje sistemi menaxhimi i punonjesve te kompanise

Nderfaqe e thjeshte qe shfaq listen e punonjesve dhe te dhenat e tyre, e pajisur me nje search bar

---

## Nevojiten

- Git
- Python (14+)
- Django (6+)

---

## Instalimi

-Klononi repo-n: https://github.com/Kristian-osoja/School_project
(git clone https://github.com/Kristian-osoja/School_project     <---Ne terminal)
-Hyni ne folder
(cd https://github.com/Kristian-osoja/School_project)
-Krijoni nje virtual environment
(python -m venv venv)
-Aktivizojeni
(.\venv\Scripts\activate)
-Instaloni Requirements
(pip install -r requirements.txt)
-Kryeni migrimet
(python manage.py migrate)
-Aktivizoni serverin
(python manage.py runserver)
-Hapni http://127.0.0.1:8000 ne web browser
